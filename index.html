How it works

It matches the pixels, not just the edges.

Stitch finds where one screenshot ends and the next begins by comparing the actual content — then trims the overlap so the join is invisible.

  • 1

    Capture a scroll

    Screenshot a long page in a few overlapping shots — Photos or the share sheet.

  • 2

    Stitch finds the seam

    It locates the matching row between shots and cuts exactly there.

  • 3

    Chrome removed once

    Status bars, nav bars and floating buttons that repeat are kept a single time.
